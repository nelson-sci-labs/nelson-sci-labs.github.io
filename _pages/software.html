ShinyLink

ShinyLink is an R Shiny web app that makes data de-duplication and record linkage insanely easy. ShinyLink offers a highly intuitive workflow via a an easy-to-use, user-friendly graphical user interface connected to the fastLink R package at the back-end.

QC-Normalization Tools - Shiny

An R-Shiny interface to QC normalization Tools, mainstream QC based normalization (PQN, QC-QLSC) are included as backend service for quality Control of massspectrometry based omics data (metbaolomics, lipidomics, proteomics).

Targeted metabolomics data Calibration - Shiny

An R-Shiny based tool that providing method for determining the concentration of a substance in an unknown sample by comparing the unknown to a set of standard samples of known concentration.

CMMR: CEU Mass Mediator RESTful API

CEU (CEU San Pablo University) Mass Mediator is an on-line tool for aiding researchers in performing metabolite annotation. 'cmmr' (CEU Mass Mediator RESTful API) allows for programmatic access in R: batch search, batch advanced search, MS/MS (tandem mass spectrometry) search, etc.
